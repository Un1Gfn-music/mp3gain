分析メニュー

分析 メニューには下記の項目があります。 これらはファイルリスト中での右クリックでも利用できます。 (注記):

トラック分析 (Ctrl-T) 各ファイルを個別に分析し、ファイルの詳細をリストに表示します。 リストの各項目についての情報は、 ファイルリスト セクションを見てください。
アルバム分析 (Ctrl-U) アルバム 中の各ファイルを全体の一部分として分析し、ファイルの詳細をリストに表示します。 リストの各項目についての詳細は、 ファイルリスト セクションを見てください。
最大無クリップ分析 (Ctrl-M) ( 最大化 機能を有効にしている時のみ表示されます )
ファイルの音量を最大化したいだけならば、この機能で、ファイルが人間の耳に実際にどんな大きさで聞こえるかを判定する  Replay Gain の計算に余計な時間を費やすこと無く、リスト中の各MP3ファイルのピーク音量を見つけられます。
分析のクリア ファイルリスト.中の分析結果をクリアします。 ファイルはリスト中に残りますが、各項目の情報は削除されます。

(注記) - 右クリックのポップアップ・メニューを使う場合は、MP3Gain は ファイルリスト.で現在選択されているファイルに対してのみ、選ばれた機能を実行します。
