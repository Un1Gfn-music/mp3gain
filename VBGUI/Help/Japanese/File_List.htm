ファイルリスト

ファイルリストは、処理するために追加された全ファイルを表示します。 もしファイル名が 赤色 になっている場合、下記の内の1つ以上が当てはまります：
  • ファイルが現在クリップしている (「クリッピング」列の "Yes " で示されます)
  • トラックゲインを適用すると、クリップが発生する (「クリップ (トラック)」列の "Yes " で示されます)
  • アルバムゲインを適用すると、クリップが発生する (「クリップ (アルバム)」列の "Yes " で示されます)

ファイルリストの各列の意味は次の通りです：
音量 MP3の現在の音量が、dB SPLで表示されます。 正確に基準に合わせて調整されたシステムでMP3を再生するならば、この平均音圧レベル（dB）になります。
クリッピング MP3ファイルが現在クリップしていると、この列に Yes が表示されます。 「クリッピング」とは、MP3ファイルがプレイヤーによりデコードされる時、幾つかの音声サンプルの音が大きすぎる事を意味します。プレイヤーはこれらのサンプルが最大許容値を超えないように、これらを「クリップします」（切り詰める、刈る、などの意）。このクリッピングは、曲の大音量部分で一種の不快な「ガリガリいう」音を発生させます。
トラックゲイン 平均音量が目標音量になるべく近くなるようにするために、MP3ファイルに適用すべきdB変更量。
クリップ (トラック) 算定されたトラックゲインを適用するとMP3がクリップする場合に、この列に Yes が表示されます。
無クリップ最大ゲイン ( 最大化機能が有効な時だけ表示されます)
MP3ファイルがクリッピングしない範囲で適用できる、最大のゲイン変更量を表示します。
アルバム音量 現在のアルバム全体の音量です。
アルバムゲイン アルバム全体の平均音量が目標音量になるべく近くなるようにするために、アルバム 中の全MP3ファイルに適用すべきdB変更量。
クリップ (アルバム) 算定されたアルバムゲインを適用するとMP3がクリップする場合に、この列に Yes が表示されます。

注記： - トラックゲインとアルバムゲイン列は、最も近い 1.5 dB 単位の値に丸められます。 MP3GainがMP3ファイルの修正に使用する 無劣化な方法 では、音量変更可能な最小量は 1.5 dB に制限されます。

ファイルのクリッピング列に ??? が表示される場合、そのMP3はおそらく内部に若干の悪いデータを含んでいると思われます。この悪いデータは、ダウンロードエラー、2つのMP3を繋いで1つにする際の稚拙な作業、または他の要因による可能性があります。そのMP3をじっくり聞けば多分どこかで、大きなクリック音、飛び、チッという音、或いは他の異音を聞くことが出来るでしょう。この悪いデータが最大のクリッピングを引き起こしているので、プログラムは最大振幅を正確に決定することが出来ません。
幸いにも、殆どの場合は悪いデータの量は十分に少ないので、ファイル全体の音量計算には影響を及ぼしません。 算出されたトラックとアルバムゲイン値は、多分まだ妥当な値でしょう。
